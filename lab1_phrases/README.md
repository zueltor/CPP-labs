# Helloworld sample project!

## Build

```
$ cd build
$ cmake ..
$ make
```

## Run

```
$ ./phrases
$ ./phrases <filename> -m <number> -n <number>
```

## Test

```
$ ./unittest
===============================================================================
All tests passed (18 assertions in 2 test cases)
```
