# Drawer project!

## Build

```
$ cd build
$ cmake ..
$ make
```

## Run
```
$ ./drawer
$ ./drawer <input> -o <output>
```

## Test

```
$ ./unittest
===============================================================================
All tests passed (48 assertions in 5 test cases)
```
